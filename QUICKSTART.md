## 🟢 METHOD 1: FASTEST (30 seconds)
### Netlify Drop - Drag & Drop

**Step 1:** Put all files in one folder
```
delfi-tech-website/
├── index.html
├── about.html
├── services.html
├── customers.html
├── certifications.html
├── contact.html
└── style.css
```

**Step 2:** Go to https://app.netlify.com/drop

**Step 3:** Drag your folder onto the page

**Step 4:** Wait 30 seconds... DONE! ✓

You get a URL: `https://delfi-tech-website.netlify.app`

---

## 🟢 METHOD 2: BEST (5 minutes)
### GitHub + Netlify - Auto Updates

**A. Create GitHub Repository**

1. Go to https://github.com/signup
2. Create account
3. Go to https://github.com/new
4. Name it: `delfi-tech-website`
5. Click "Create repository"
6. Upload all 7 files
7. Click "Commit changes"

**B. Deploy to Netlify**

1. Go to https://app.netlify.com
2. Click "Sign up" → "GitHub"
3. Authorize with GitHub
4. Click "New site from Git"
5. Select your repository
6. Click "Deploy site"
7. Wait for deployment...

**DONE!** Your site is live!

**Bonus:** Every time you update files in GitHub, your site automatically updates!

---

## 🟢 METHOD 3: FOR DEVELOPERS
### Command Line

```bash
# Install Netlify
npm install -g netlify-cli

# Go to your folder
cd path/to/delfi-tech-website

# Login to Netlify
netlify login

# Deploy!
netlify deploy --prod
```

Done! You get your URL and your site is live.

## 🎨 CUSTOMIZE YOUR SITE (Optional)

### Change Colors
1. Open `style.css`
2. Find `:root` section (top)
3. Change color codes:
   - `--primary-color: #1D9E75;` ← Main color
   - `--primary-dark: #0F6E56;` ← Dark color
4. Save and redeploy

### Update Company Info
Find and replace in all HTML files:
- `Delfi Tech` → Your company name
- `info@delfitech.com.hk` → Your email
- `+852 2635 6389` → Your phone
- Address information

### Add Your Logo
1. Save logo as `logo.png`
2. Add to HTML files:
   ```html
   <img src="logo.png" alt="Company Logo">
   ```

## 🌐 CONNECT YOUR OWN DOMAIN (Optional)

1. Register domain (GoDaddy, Namecheap, etc.)
2. Go to Netlify dashboard
3. Click "Domain Management"
4. Click "Add custom domain"
5. Enter your domain: `delfitech.com`
6. Update DNS records (Netlify will show you how)
7. Wait 24-48 hours for DNS to work
8. Your domain is now live! ✓
